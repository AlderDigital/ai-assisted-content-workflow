# AI Assisted Content Workflow

A practical, vendor-agnostic reference for turning raw ideas into publishable content using repeatable steps and AI-supported workflows.

This repository exists as a **public operational reference**.  
The full explanations, examples, and FAQs live on the Alder Digital website and should be treated as the canonical source.

## What this repository contains

- A clear, 6-stage AI-assisted content workflow
- Lightweight operational checklists for review, localisation, and video dubbing
- Plain-language guidance focused on process clarity, not tools or vendors

## The workflow stages

1. Idea capture  
2. Content creation  
3. Review and approval  
4. Translation and localisation  
5. Publishing and distribution  
6. Reuse and repurposing  

Detailed breakdowns of each stage are documented in the workflow folder.
